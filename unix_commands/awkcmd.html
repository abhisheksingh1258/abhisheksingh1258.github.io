AWK Examples

  1. To run the script in sum_column file and displays the sum of the 5th column in the input_file.

    2. #!/usr/bin/awk -f
    BEGIN {sum=0}
    {sum=sum+$5}
    END {print sum}

  2. This awk command checks for the string "t4" in the 9th column and if it finds a match then it will print the entire line
    3. awk '{ if($9 == "t4") print $0;}' input_file
  3. This will print the squares of first numbers from 1 to 5.
    4. awk 'BEGIN { for(i=1;i<=5;i++) print "square of", i, "is",i*i; }'
  4. This will display the number of columns in each row
    5. awk '{print NF}' input_file
